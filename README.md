# MX-WEBSCAN
Es un escáner de vulnerabilidades web automatizado enfocado en el OWASP Top 10, diseñado para pentesters y administradores de sistemas en entornos hispanohablantes. La herramienta permite detectar fallos críticos como SQL Injection (SQLi), Cross-Site Scripting (XSS), Local File Inclusion (LFI) y Server-Side Request Forgery (SSRF)
